# 🚀 איך להתחיל - The Bill

## ⚠️ חשוב מאוד!

**אל תפתח את `index.html` ישירות!**

האפליקציה זקוקה לשרת שירוץ ברקע.

---

## 🎯 הדרך הנכונה להריץ:

### Windows:

#### דרך 1 (מומלץ) - לחיצה כפולה
```
הפעל: run.bat
```
זה יפתח אוטומטית את הדפדפן בכתובת הנכונה.

#### דרך 2 - ידנית
1. פתח טרמינל
2. הרץ:
   ```bash
   python app.py
   ```
3. פתח דפדפן ב: **http://localhost:10000**

---

## ✅ איך לדעת שזה עובד:

אמורה להיות **כתובת בדפדפן:**

http://localhost:10000  ✅ נכון!

## 🐛 פתרון בעיות

### הדף נראה בלי עיצוב / כפתורים לא עובדים

**סיבה:** פתחת את הקובץ ישירות במקום דרך השרת

**פתרון:**
1. סגור את הטאב
2. ודא שהשרת רץ (בטרמינל)
3. פתח: **http://localhost:10000**

### השרת לא עולה

**סיבה:** פורט 10000 תפוס

**פתרון:**
```bash
# עצור תהליך קודם
taskkill /F /IM python.exe

# הפעל מחדש
python app.py
```
